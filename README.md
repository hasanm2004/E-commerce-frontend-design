Ecommerce Frontend Design

This project is a desktop-based eCommerce frontend website developed as part of an internship task. It focuses on implementing a clean and pixel-perfect UI using HTML, CSS, and JavaScript.

Features:

Home Page:

Hero section with banner and call-to-action button
Product categories (Shoes, Watches, Clothes etc)
Featured products section
Product Listing Page:

Grid-based product layout
Product cards with image, name, price, and button
Load More functionality
Product Detail Page:

Product image, name, price, and description
Size/option selection dropdown
Add to Cart button
Basic JavaScript interactivity
Technologies Used:

HTML5 (Semantic structure)
CSS3 (Flexbox, layout, styling)
JavaScript (Basic interactivity)
Project Structure: ecommerce-frontend-design/ │ ├── index.html ├── products.html ├── productdetail1.html ├── productdetail2.html ├── css/ │ └── style.css ├── js/ │ └── script.js ├── Images/
